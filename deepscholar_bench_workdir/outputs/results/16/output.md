# Related Work

## Open-Vocabulary Object Detection

Open-vocabulary object detection (OVD) generalizes closed-set detectors to arbitrary categories specified in natural language at inference time. The paradigm was introduced by OVR-CNN [1](https://arxiv.org/abs/2011.10678), which pre-trains a visual encoder on image–caption pairs before fine-tuning on a limited set of base classes and transferring to novel ones. ViLD [2](https://arxiv.org/abs/2104.13921) distills the embeddings of a pretrained vision–language model (CLIP [3](https://arxiv.org/abs/2103.00020)) into a two-stage detector so that region features align with textual class prototypes. RegionCLIP [4](https://arxiv.org/abs/2112.09106) extends this idea by explicitly pretraining region–text alignment on image–caption data, while Detic [5](https://arxiv.org/abs/2201.02605) leverages large-scale image-level supervision from classification datasets to scale detection to more than twenty thousand classes. F-VLM [6](https://arxiv.org/abs/2209.15639) shows that competitive OVD can be obtained by simply building a detector on top of a frozen vision–language backbone, avoiding costly re-alignment.

Another line of work casts detection as a phrase-grounding problem. GLIP [7](https://arxiv.org/abs/2112.03857) unifies object detection and grounding by treating class names as language prompts, enabling training on both detection and grounding data. DetCLIP [8](https://arxiv.org/abs/2209.09407) enriches this formulation with a concept dictionary that provides definitions for rare categories. OV-DETR [9](https://arxiv.org/abs/2203.11876) adapts DETR-style architectures to open-vocabulary conditions via conditional matching between queries and class embeddings, and Grounding DINO [10](https://arxiv.org/abs/2303.05499) further couples DINO-based detection with grounded language pre-training, becoming a strong open-set baseline. Transformer-based OWL-ViT [11](https://arxiv.org/abs/2205.06230) and its self-training successor OWLv2 [12](https://arxiv.org/abs/2306.09683) demonstrate that plain ViT backbones with contrastive image–text pre-training can scale to billions of pseudo-labeled boxes. More recently, YOLO-World [13](https://arxiv.org/abs/2401.17270) integrates open-vocabulary heads into an efficient YOLO backbone, providing the real-time inference speed on which our plug-and-play adapter is designed to build.

## Prompt Engineering and Vocabulary Adaptation

Because vision–language detectors are sensitive to the wording and scope of the input vocabulary, several works focus on adapting the textual side of the pipeline. CoOp [14](https://arxiv.org/abs/2109.01134) learns continuous prompt vectors that replace hand-crafted templates when transferring CLIP to downstream classification tasks. DetPro [15](https://arxiv.org/abs/2203.14940) extends learnable prompting to detection, optimizing category context tokens for open-vocabulary detectors. PromptDet [16](https://arxiv.org/abs/2203.16513) further shows how uncurated web images can be exploited to mine prompts that expand the vocabulary reachable by a detector. Unlike these approaches, our Vocabulary Adapter is training-free and refines the *set* of active categories per image rather than the wording of individual prompts, complementing prompt-tuning methods.

## Image Captioning for Downstream Perception

Our adapter relies on an image captioner to enumerate what is actually present in the scene before pruning the user vocabulary. BLIP [17](https://arxiv.org/abs/2201.12086) and BLIP-2 [18](https://arxiv.org/abs/2301.12597) provide strong open-source captioners that jointly train or align image encoders with large language models, producing fluent descriptions of visible objects. Using such captions as intermediate semantic supervision echoes the caption-driven training signal exploited by OVR-CNN [1](https://arxiv.org/abs/2011.10678) and RegionCLIP [4](https://arxiv.org/abs/2112.09106), but we use captions at *inference time* to filter candidate categories rather than to learn region–word alignment.

## Benchmarks

We evaluate on the widely-used COCO benchmark [19](https://arxiv.org/abs/1405.0312), which established the standard object detection protocol on common everyday categories, together with the larger and more diverse Objects365 dataset, following the evaluation practice adopted by recent OVD works such as GLIP [7](https://arxiv.org/abs/2112.03857), Grounding DINO [10](https://arxiv.org/abs/2303.05499), and YOLO-World [13](https://arxiv.org/abs/2401.17270). These benchmarks let us verify that VocAda consistently improves state-of-the-art detectors across both moderate- and large-vocabulary regimes.
